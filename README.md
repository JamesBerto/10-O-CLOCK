This project is 98% vibe-coded, meaning it was built with a focus on creative flow, intuition, and aesthetic-driven development rather than strict structure from the start.
It serves as an experimental test to explore how “vibe coding” works in practice—prioritizing feel, design, and rapid iteration over traditional coding methods.
The main purpose of this project is to act as a platform for advertising and promoting social media channels, showcasing content in a visually engaging and modern way.
It emphasizes user experience, smooth navigation, and an appealing interface to capture attention and drive engagement across different social platforms.
